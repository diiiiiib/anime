# kaneki_ken - موقع مشاهدة الأنمي

موقع إلكتروني كامل لمشاهدة الأنمي، مبني باستخدام HTML و CSS و JavaScript (Vanilla JS فقط بدون أطر عمل).

## 📋 المتطلبات

- متصفح حديث يدعم HTML5 و CSS3 و JavaScript ES6
- خادم محلي (مثل Live Server) لتشغيل الموقع (اختياري)

## 🚀 التشغيل

1. قم بتحميل المشروع
2. افتح ملف `index.html` في متصفحك
3. استمتع بمشاهدة الأنمي!

## 📂 هيكل المشروع

```
kaneki_ken/
│
├── index.html          # الصفحة الرئيسية
├── anime.html          # صفحة المسلسلات
├── movie.html          # صفحة الأفلام
├── details.html        # صفحة تفاصيل الأنمي
├── watch.html          # صفحة مشاهدة الحلقات
│
├── css/
│   └── style.css       # ملف الأنماط الرئيسي
│
├── js/
│   ├── data.js         # ملف البيانات (يحتوي على جميع الأنميات)
│   ├── main.js         # الوظائف الرئيسية للموقع
│   ├── details.js      # وظائف صفحة التفاصيل
│   └── watch.js        # وظائف صفحة المشاهدة
│
└── assets/
    ├── images/         # مجلد الصور
    └── icons/          # مجلد الأيقونات
```

## 📦 إضافة أنميات جديدة

لإضافة أنمي جديد، اتبع الخطوات التالية:

1. افتح ملف `js/data.js`
2. أضف كائن جديد إلى مصفوفة `animes` بالتنسيق التالي:

```javascript
{
  id: 6,  // معرف فريد للأنمي
  title: "اسم الأنمي",
  type: "series",  // series أو movie
  cover: "assets/images/cover.jpg",  // مسار صورة الغلاف
  background: "assets/images/background.jpg",  // مسار صورة الخلفية
  description: "وصف الأنمي",
  genre: ["أكشن", "غموض"],  // التصنيفات
  rating: "8.5",  // التقييم
  year: "2023",  // سنة الإصدار
  episodes: [  // قائمة الحلقات
    {
      number: 1,  // رقم الحلقة
      title: "عنوان الحلقة",
      video: "https://example.com/video.mp4"  // رابط الفيديو
    }
  ]
}
```

3. احفظ الملف
4. سيتم إضافة الأنمي تلقائياً إلى الموقع

## 🎨 المميزات

- تصميم متجاوب يعمل على جميع الأجهزة
- دعم الوضع الداكن
- بحث عن الأنميات

- تحميل بطيء للصور لتحسين الأداء
- مشاركة الحلقات على وسائل التواصل الاجتماعي
